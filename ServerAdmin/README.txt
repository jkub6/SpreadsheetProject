Spreadsheet Admin Site
-Login Page
Enter Username
Enter Password
Enter ServerIP/Website to connect to

In event of failed password or failed server connect, page will inform and send back to login
Responsibility of the user to identify valid website

Spreadsheet List Page - Shows list of spreadsheets, allows admin to delete spreadsheets
Can redirect to list of users, add more spreadsheets

In adding more spreadsheets, user can add one more or add another and then return.
In event someone else makes edit, page will  inform that the old user needs to be deleted or to create a different user.

If a spreadsheet has already been deleted, will also inform user

User List Page - Shows List of users, allows admin to delete users
Can redirect to the spreadsheet list or add more users

Similar functions to spreadsheet list and will make sure that the current values happen

Signing out logs out the user
Website forces relogin after 5 minutes
